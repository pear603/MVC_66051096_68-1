ใช้ภาษา python ร่วมกับ html และ css ผ่าน Flask API
ออกแบบตามคอนเซปต์ MVC ที่ประกอปไปด้วย Model View Controller ทำงานแยกหน้าที่กันชัดเจน
Model มีหน้าที่ระบุ Business Logic สิ่งที่ระบบสามารถทำได้
View มีหน้าที่แสดงผลการทำงาน
Controller มีหน้าที่ควบคุมการทำงานระหว่าง View และ Model โดยจัดการกับ input ของผู้ใช้และนำไปใช้กับ Model ที่ระบุไว้ซึ่งนำผลลัพธ์ไปให้ View ใช้แสดงผล

ซึ่งได้ทำการออกแบบตามโจทย์ที่อ้างอิงจาก MVC Architecture ดังนี้
